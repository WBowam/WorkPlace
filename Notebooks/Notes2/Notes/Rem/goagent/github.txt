====== github ======
Created 星期六 31 八月 2013

__上传__
接下来在本地仓库里添加一些文件，比如README，	
$ git add README
$ git commit -m "first commit"

上传到github：
连接
$ git remote add origin git@github.com:yourName/yourRepo.git
	
$ git push origin master
